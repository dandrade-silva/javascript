Operadores do JavaScript para Iniciantes

  • + : Adição
  • - : Subtração
  • * : Multiplicação
  • / : Divisão
  • % : Resto da divisão (Módulo)
  • ** : Exponenciação (disponível a partir do ECMAScript 2016)
  • = : Atribuição simples
  • += : Atribuição com adição
  • -= : Atribuição com subtração
  • *= : Atribuição com multiplicação
  • /= : Atribuição com divisão
  • %= : Atribuição com resto da divisão
  • **= : Atribuição com exponenciação
  • == : Igual a (compara valor)
  • != : Diferente de (compara valor)
  • === : Igual a (compara valor e tipo)
  • !== : Diferente de (compara valor e tipo)
  • > : Maior que
  • < : Menor que
  • >= : Maior ou igual a
  • <= : Menor ou igual a
  • && : AND lógico (E)
  • || : OR lógico (OU)
  • ! : NOT lógico (NÃO)
  • ++ : Incremento
  • -- : Decremento

Ordem de Precedência dos Operadores do JavaScript

A ordem de precedência determina a ordem em que os operadores são avaliados em uma expressão.

  1. () - Parênteses
  2. ** - Exponenciação
  3. *, /, % - Multiplicação, Divisão e Resto da Divisão
  4. +, - - Adição e Subtração
  5. <, >, <=, >= - Operadores de Comparação
  6. ==, !=, ===, !== - Operadores de Igualdade e Desigualdade
  7. && - AND Lógico
  8. || - OR Lógico
  9. = - Atribuição

É possível alterar a ordem de avaliação usando parênteses para controlar a precedência dos operadores.
